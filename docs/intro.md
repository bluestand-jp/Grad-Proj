# チュートリアル概要

こんにちは！
ここはUniversity eSports Kansaiのドキュメントです。

この概要で、５分で全体像をとらえてみましょう。

## はじめに

まず、イベントを制作するには**強靭な精神力**が必要です。

あなたが生半可な覚悟でこの制作マニュアルを読んでいるのであれば、少し考え直してみましょう。思ったよりも道のりは長く、険しいものとなるでしょう。

それでも制作を進めてやる！というGod Mentalをお持ちなら、この先へ進みましょう。

### 必要なもの

イベントには必要なものがたくさんあります。

その中でも最初に用意するものは以下です。

- 仲間
- やりたいイベントの像
- 言葉では言い表せないくらいのモチベーション
