## このセクションについて

イベントを成功させるためには、しっかりとした準備が不可欠です。準備段階は大会運営において**最も長く、最も重要なフェーズ**です。

---

## 📅 スケジュール概要

| フェーズ | 目安期間 | 本番からの逆算 |
|---------|----------|---------------|
| 概要明確化 | 約2週間 | 本番 **10〜12週間前** |
| 内容制作 | 約3週間 | 本番 **7〜10週間前** |
| 大会形成 | 約3〜5週間 | 本番 **2〜7週間前** |

:::tip[日程例：3ヶ月前スタートの場合]
- **12週間前**: 目的設定・ターゲット決定・タイトル選定
- **10週間前**: 大会ルール策定・デザイン制作開始
- **8週間前**: 募集要項完成・SNS告知開始
- **6週間前**: エントリー受付開始
- **4週間前**: エントリー締切・トーナメント作成
- **2週間前**: 進行マニュアル完成・リハーサル準備
:::

## 📝 各フェーズの詳細

### 概要明確化（約2週間）

大会の「なぜ」「どうして」「いつ」「どこで」「なにを」を明確にするフェーズです。

| タスク | 内容 |
|-------|------|
| 目的設定 | 学生同士の交流、地域活性、スポンサー訴求など |
| ターゲット決定 | 大学生限定 or オープン参加 |
| タイトル選定 | VALORANT、LoL、スプラトゥーンなど |
| 規模・形式 | オンライン/オフライン、参加チーム数、開催期間 |

### 内容制作（約3週間）

大会開催に必要なディテールを制作するフェーズです。

| タスク | 内容 |
|-------|------|
| ルール策定 | 出場条件、禁止事項、マップ・キャラルール |
| 募集要項作成 | エントリーフォーム、応募条件 |
| デザイン制作 | ロゴ、キービジュアル、SNS素材、配信画面 |
| ツール準備 | Discord、応募フォーム、配信設定 |

### 大会形成（約3〜5週間）

エントリーを募り、大会としての形を成していくフェーズです。

| タスク | 内容 |
|-------|------|
| 参加チーム確定 | エントリー締切後の確認・リスト化 |
| トーナメント作成 | 抽選・シード順決定 |
| 進行マニュアル | タイムテーブル、台本、役割分担 |
| リスク管理 | トラブル時の対応フロー整備 |
