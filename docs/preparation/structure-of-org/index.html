組織形成について

大会を運営するにあたって、効率的な組織を構築するということは、プロジェクト進行にとってクリティカルな問題になります。
そのため、組織形成は慎重に行う必要があります。以下に組織形成の例を添え、基本事項を記します。

1. 本部（プロジェクトマネジメント）​

大会全体の中心となるチームです。

主な役割

  • 団体全体の進行管理（いわばプロジェクトマネージャー）
  • 全チームの進捗確認、連携調整
  • 大会全体の構成をまとめ、台本・香盤表（当日の動きの表）を作成
  • トラブル発生時の最終判断・指示出し

ポイント

  • 最も情報が集まるポジション。制作物・会場・配信・競技ルールなど、すべてを把握する必要がある。
  • 1〜3名程度で構成することが多い。

2. 配信チーム​

「esports大会＝配信あり」が基本であるため、質の高い配信は大会の満足度を大きく左右します。

主な役割

  • 配信機材の選定と準備
  • OBSや配信ソフトの設定
  • カメラ・マイクなどのチェック
  • 試合進行に合わせた映切り替え
  • 当日の配信監視・トラブル対応

ポイント

  • 事前リハーサルは必須。回線テストも忘れずに。
  • 映像班・音響班など細分化することも可能。

3. 営業チーム（スポンサー・資金調達）​

大会のクオリティを高めるためには資金が必要です。学生大会では、少額でもスポンサーが入るだけで大会の幅が大きく広がります。

主な役割

  • スポンサー獲得（企業・団体・学校など）
  • 协賛メニューの作成（ロゴ掲載、配信内宣伝、ブース出展など）
  • 資金管理・予算配分
  • 会場費や賞金・景品の調整
  • 企業との連絡窓口

ポイント

  • スポンサー資料（提案書）は丁寧に作ること。
  • 企業との連絡はスピードと丁寧さが最重要。

4. 選手管理チーム​

参加者が気持ちよくプレイできる環境を整えるチームです。

主な役割

  • エントリー受付と問い合わせ対応
  • 選手向け案内（持ち物、ルール、当日の動線など）
  • 試合当日の受付・誘導
  • 設備トラブル対応（デバイス問題、ネットワーク問題など）

ポイント

  • 選手目線での運営は、満足度を大きく左右する。
  • 競技チームと密に連携することが大切。

5. 競技チーム（レギュレーション・大会形式）​

公平な大会を成立させるための柱となるチーム。

主な役割

  • ルールブックの作成
  • 大会形式の決定（トーナメント・リーグ・BO1/BO3 など）
  • 使用デバイス・設定・禁止事項の明確化
  • レフェリー（審判）の担当
  • 試合進行の管理（試合開始、対戦組み合わせ、結果報告）

ポイント

  • 不正やルール違反が起きないよう事前に細かく規定しておく。
  • オフライン大会の場合、トラブル時の裁定ルールも必須。

6. クリエイティブチーム（デザイン・広報）​

大会の印象を大きく左右する「見える部分」を担当するチーム。

主な役割

  • SNS投稿デザイン、ポスター、バナーの制作
  • Webサイトや大会エントリーページの制作
  • PR動画・ハイライト動画の作成
  • ブランディング統一（色・フォント・ロゴの管理）

ポイント

  • 投稿の統一感を揃えることで大会のブランド力が上がる。
  • 配信チームとの連携で、配信画面のデザインも担当可能。
