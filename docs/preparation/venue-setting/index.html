会場準備

ここではオフラインで大会を行う上で特に大事となる会場手配について話します。

会場のファーストインプレッションが観客・来場者に与える印象は、とても重要です。
その大会の色を決め、特徴やブランドパーソナリティを大きく表現するものだからです。

そんな直接的な表現法である「大会制作」には、細心の注意を要します。

会場手配の手順​

大会運営において、会場手配は最も重要な工程のひとつです。

適切な会場を確保することにより、当日の進行が円滑になるだけでなく、参加者や観客に対して安心感を与えることができます。

まず、会場選定では大会の規模・内容・必要設備を明確にすることが重要です。
PCを使用するLAN大会か、ステージイベントを伴うオフラインイベントかによって、必要な電力容量や通信環境、席数、控室の有無が大きく異なります。
また、大学施設を利用する場合は、利用申請期日が定められていることが多いため、早期の調整と申請が必須です。

次に、候補となる会場に対して 下見（ロケハン） を行い、実際の導線や設備状況を確認します。
特に、配信機材の設置スペース、電源の位置、騒音環境、ネットワーク速度・安定性などは事前確認が欠かせません。また、当日の運営スタッフや選手がスムーズに動けるよう、控室や荷物置き場の確保も行います。

最終的に、使用料、設備費、キャンセルポリシーなどの条件を比較し、最も適した会場を選定します。
会場決定後は、利用許可書や鍵の受け渡し方法、当日の入館手順などの細かな点まで確認し、運営チーム全体で共有しておくことが重要です。

1. 要件整理​

  • 会場を探し始める前に、イベントに必要な条件を明確にする。
  • 参加人数（選手／観客／運営スタッフ）
  • イベント形式（大会／ステージイベント／展示）
  • 必要設備（机・椅子・プロジェクター・モニター・スクリーン）
  • 必要電力（PC台数 × 消費電力）
  • 有線LANの必要本数
  • 控室・荷物置き場の必要性
  • 驚車場・搬入経路の有無

2. 候補会場のリストアップ​

  • 大学施設（講義室、ホール、学生会館）
  • 地域の公共施設（市民センター、会議室）
  • 民間イベントスペース
  • eスポーツ施設（貸し切り可否を確認）

問い合わせ項目​

  • 料金（時間貸し・終日）
  • 利用可能時間
  • 禁止事項（飲食、装飾、持ち込み機材など）
  • ネットワーク設備
  • 電源容量

3. 会場の下見（ロケハン）​

現地確認では、以下のチェックが必須。

  • 設備チェック
  • 電源の数・位置
  • LANポートの位置
  • 無線LANの強度・安定性
  • 照明の明るさと調整の可否
  • 机・椅子の数
  • 音響設備の有無と接続端子
  • 動線確認
  • 選手や観客の導線
  • 運営席の位置
  • カメラ・配信機器の設置スペース
  • 機材搬入口と最短ルート
  • スタッフの休憩・荷物置きスペース
  • トラブルリスク確認
  • 周辺の騒音の可能性
  • 冷暖房設備
  • コンセント不足への対応（延長コード可否）

4. 会場の仮予約・申請​

  • 大学施設 → 利用申請書の提出、学生団体としての許可
  • 民間施設 → 見積書の確認、予約金の有無
  • 支払い方法の確認（当日現金・事前振込など）

※キャンセルポリシーは必ず事前に共有する。

5. レイアウト設計と必要機材の洗い出し​

  • 会場図面に、机・椅子・PC・配信席などの配置を書き込む
  • 電源タップ・LANケーブルの長さを算出
  • カメラ位置や配信機材の導線も図示
  • 導線が狭くならないよう調整

6. 会場側との最終確認​

  • 鍵の受け渡し方法
  • 当日の入館手順
  • スタッフの人数
  • 使用する設備の事前申請は完了しているか
  • 延長ケーブルやLANケーブルの持ち込み許可

会場手配は、早ければ早いほど良い。

最低でも1〜2か月前に完了しておくのが望ましい。
