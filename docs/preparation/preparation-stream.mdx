# esportsの大会配信について

esportsの大会において、「大会＝配信」 と言っても過言ではありません。
対人ゲームの多くがオンラインで行われるため、実際に会場に来られないファンは オンライン配信を通して大会を楽しみます。
だからこそ、esports大会において“配信”のクオリティは大会全体のイメージを大きく左右します。

ただし、「esportsだから特別な映像演出が必要なのか？」と言われれば、実はそうでもありません。
一番大切なのは “見てて飽きないこと” です。

大切なのは「視聴者が見やすいかどうか」

配信は常に 視聴者ファースト で作られるべきです。

・見やすい画面構成

・同じ映像が続かない工夫

・どのタイミングで配信を開いても状況が理解できる構成

・必要な情報が分かりやすく画面に表示されていること

選手やスタッフがゲームに真剣に向き合っているのと同じように、
配信を見て応援してくれる視聴者もまた、大会に熱量を持って参加しています。
その熱に応えるためにも、飽きさせない配信 が何より重要です。

【配信をする上で準備すること】

ここからは、配信チームが大会のために行う準備について詳しく説明します。

① 制作チームとのすり合わせ（＝最重要）

e-sportsイベントでは、
「イベント制作班」 と 「配信・技術班」 は別であることがほとんどです。

制作班
→「こういう画を見せたい」「こういう演出をしたい」という“理想・要望”を持っている

配信・技術班
→ その“理想”をどう実現するかを考え、技術面で形にするプロ

例えば…

「選手がプレイしている姿を視聴者に届けたい」
　→ 選手席にカメラを設置して配信にのせる検討をする

「スーパープレイをリプレイで見せたい」
　→ リプレイ専用のPCやキャプチャを追加し、配線に組み込む

この「やりたいこと」を丁寧に拾い上げる段階がもっとも重要で、
大会の完成度がここで決まると言っても過言ではありません。

② 配線図制作

制作側とのすり合わせが終わったら、
次は 配線図（システム図） の制作に進みます。

・使用する機材の洗い出し

・会場のサイズ

・電源位置

・ケーブルの通り道

・映像・音声の流れの確認

これらを踏まえて、最適な配線ルートを考えて図にしていきます。

実はこの作業、

地味・時間がかかる・シンプル

……ですが、
大会配信でトラブルを起こさないための最重要工程 です。

配線図を作る途中で、
「ここはもっと効率的な配線ができる」
「この機材の位置は変更した方がよい」
と改善点に気づくことも多く、良い配信に直結します。

慣れてくるとパズルゲームのようで楽しい工程でもあります。

③ 当日の機材仕込み

配線図が完成したら、いよいよ大会当日の 機材仕込み（セッティング） に入ります。

・モニター・PCの設置

・スイッチャー / キャプチャの配線

・カメラの位置・フォーカス調整

・音声機材の設置

・配信用ソフトの設定（OBS等）

当たり前ですが、事前準備が多ければ多いほど、当日の作業はスムーズになります。

配信仕込みは チームワークが非常に重要なセクション です。

・ケーブルを持つ人

・機材を設置する人

・音声を確認する人

・映像をチェックする人

など役割が細かく分かれているので、
円滑なコミュニケーションとスピード感が求められます。

独特の緊張感と集中力が漂う一方で、
本番前の「1つのチームとしての信頼感」が強く感じられる瞬間でもあります。
