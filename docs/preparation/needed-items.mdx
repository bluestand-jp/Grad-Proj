## このセクションについて

イベント運営には、さまざまな機材やツールが必要です。このセクションでは、必要なアイテムをカテゴリ別に整理し、**準備漏れを防ぐためのチェックリスト**としても活用できます。

---

## 📅 準備スケジュール

| 時期 | 準備内容 |
|-----|---------|
| 本番 **8週間前** | 必要機材のリストアップ・調達開始 |
| 本番 **4週間前** | 機材の動作確認・予備の確保 |
| 本番 **1週間前** | 機材リストの最終確認・梱包準備 |
| 本番 **前日** | 搬入・設置・最終チェック |

:::tip[レンタル機材の場合]
レンタル機材は本番1週間前までに予約完了させることをおすすめします。人気の機材は早めに押さえましょう。
:::

## 📝 詳細チェックリスト

### 配信関連

| アイテム | 用途 | 備考 |
|---------|------|------|
| 配信用PC | OBS・配信ソフト実行 | スペック要確認 |
| キャプチャーボード | ゲーム映像取り込み | 4K対応推奨 |
| Webカメラ | キャスター映像 | 1080p以上推奨 |
| マイク | キャスター音声 | ポップガード必須 |
| オーディオインターフェース | 音声管理 | 複数入力対応 |

### 競技関連

| アイテム | 用途 | 備考 |
|---------|------|------|
| ゲーミングPC | 選手プレイ用 | タイトル要件確認 |
| ゲーミングモニター | 低遅延表示 | 144Hz以上推奨 |
| 予備デバイス | トラブル時対応 | マウス・キーボード各2 |
| LANケーブル | 有線接続 | Cat6以上推奨 |
| 電源タップ | 機材電源 | 容量に注意 |
