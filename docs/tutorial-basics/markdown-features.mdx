# Markdown の機能

Docusaurus は **[Markdown](https://daringfireball.net/projects/markdown/syntax)** をサポートし、さらに便利な **拡張機能** も提供します。

## Front Matter

Markdown ドキュメントの先頭には [Front Matter](https://jekyllrb.com/docs/front-matter/) と呼ばれるメタデータを記述できます。

```text title="my-doc.md"
// highlight-start
---
id: my-doc-id
title: ドキュメントのタイトル
description: ドキュメントの概要
slug: /my-custom-url
---
// highlight-end

## Markdown の見出し

リンクを含む Markdown テキスト [リンク例](./hello.md)
```

## リンク

Markdown のリンク構文を利用できます。絶対パス・相対パスのどちらでも指定可能です。

```md
ページを作成する方法は[こちら](/create-a-page)。
```

```md
ページを作成する方法は[こちら](./create-a-page.md)。
```

**結果:** ページを作成する方法は[こちら](./create-a-page.md)。

## 画像

通常の Markdown 画像構文を利用できます。

`static` ディレクトリ（例: `static/img/docusaurus.png`）にある画像は絶対パスで参照できます。

```md
![Docusaurus ロゴ](/img/docusaurus.png)
```

![Docusaurus ロゴ](/img/docusaurus.png)

Markdown ファイルからの相対パスでも画像を参照できます。ドキュメントと画像を近くに配置できるため管理が容易になります。

```md
![Docusaurus ロゴ](./img/docusaurus.png)
```

## コードブロック

構文ハイライト付きのコードブロックをサポートしています。

````md
```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
  return <h1>こんにちは、Docusaurus！</h1>;
}
```
````

```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
  return <h1>こんにちは、Docusaurus！</h1>;
}
```

## アドモニション

Docusaurus には注意喚起やコールアウトを表示する専用記法があります。

```md
:::tip アドバイス

この便利な機能を活用してみましょう

:::

:::danger 注意

この操作は危険です

:::
```

:::tip アドバイス

この便利な機能を活用してみましょう

:::

:::danger 注意

この操作は危険です

:::

## MDX と React コンポーネント

[MDX](https://mdxjs.com/) を使うと、ドキュメントをより **インタラクティブ** にでき、Markdown 内で任意の **React コンポーネント** を使用できます。

```jsx
export const Highlight = ({children, color}) => (
  <span
    style={{
      backgroundColor: color,
      borderRadius: '20px',
      color: '#fff',
      padding: '10px',
      cursor: 'pointer',
    }}
    onClick={() => {
      alert(`色 ${color}（ラベル: ${children}）がクリックされました`);
    }}>
    {children}
  </span>
);

これは <Highlight color="#25c2a0">Docusaurus グリーン</Highlight> です！

これは <Highlight color="#1877F2">Facebook ブルー</Highlight> です！
```
