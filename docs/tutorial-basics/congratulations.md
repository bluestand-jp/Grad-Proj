# おつかれさま！

これで **Docusaurus の基本操作** を理解し、**初期テンプレート** に手を加えられるようになりました。

Docusaurus には、まだまだ活用できる機能がたくさんあります。

**あと 5 分** あるなら、**[ドキュメントのバージョン管理](../tutorial-extras/manage-docs-versions.md)** と **[多言語対応](../tutorial-extras/translate-your-site.md)** を覗いてみましょう。

このチュートリアルで **分かりにくい点** や **不具合** を見つけたら、[ぜひ報告してください](https://github.com/facebook/docusaurus/discussions/4610)。

## 次のステップ

- [公式ドキュメント](https://docusaurus.io/) を読む
- [`docusaurus.config.ts`](https://docusaurus.io/docs/api/docusaurus-config) でサイト設定を変更する
- [`themeConfig`](https://docusaurus.io/docs/api/themes/configuration) にナビバーやフッター項目を追加する
- [デザインとレイアウト](https://docusaurus.io/docs/styling-layout) をカスタマイズする
- [検索バー](https://docusaurus.io/docs/search) を導入する
- [Docusaurus Showcase](https://docusaurus.io/showcase) で事例を探す
- [Docusaurus コミュニティ](https://docusaurus.io/community/support) に参加する
