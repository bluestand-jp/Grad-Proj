## このセクションについて

これまでの準備の集大成となる大会当日。本番は**新しいことをする場ではなく、リハーサルで確認したことを落ち着いて実行する日**です。

---

## 📅 当日のタイムライン例

| 時間 | イベント | 担当 |
|-----|---------|------|
| 開場 -3時間 | スタッフ集合・最終確認 | 本部 |
| 開場 -2時間 | 機材最終チェック・仕込み完了 | 技術 |
| 開場 -1時間 | 選手受付開始 | 競技 |
| 開場 -30分 | 選手セットアップ・動作確認 | 競技 |
| 開場 | 配信開始・オープニング | 配信 |
| 本番中 | 試合進行・配信管理 | 全員 |
| 終了後 | 表彰式・エンディング | 本部 |
| 撤収 | 機材片付け・振り返り | 全員 |

:::tip[オンライン大会の場合]
- **本番2時間前**: 運営Discord集合・役割確認
- **本番1時間前**: 選手Discord確認・ロビー準備
- **本番30分前**: 配信テスト・最終確認
- **本番開始**: 試合進行・配信管理
:::

## 📝 業務の詳細

### 選手対応の流れ

| ステップ | 内容 |
|---------|------|
| 1. 受付・点呼 | 指定時間に受付、チームごとに確認 |
| 2. セットアップ | PC・デバイス接続、感度調整、ログイン |
| 3. 試合前確認 | ルール再確認、準備完了報告 |
| 4. 試合進行 | ロビー案内、試合開始の合図 |
| 5. 試合後 | 結果報告、次試合への案内 |

### キャスターサポート

| サポート内容 | 具体例 |
|------------|-------|
| カウントダウン | 「10秒前」「5秒前」「スタート！」 |
| 進行状況共有 | 「マップBAN中」「試合開始まで2分」 |
| トラブル情報 | 「選手PC不具合」「回線トラブル」 |
| 時間調整 | 「巻いてください」「トピック追加で」 |

## ⚠️ トラブル対応

:::warning[トラブルは必ず起きる]
どれだけリハを重ねても、本番でトラブルは起こりえます。大切なのは**どう対応するか**です。
:::

### よくあるトラブル

- デバイスが認識しない
- ゲーム内ログインできない
- 回線が不安定
- 選手が設定データを保存していない

### 対応の流れ

1. **どのセクションの問題か切り分ける**（技術/制作/競技/配信）
2. **担当セクションが解決案を考える**
3. **方針が決まったら全セクションに共有**
