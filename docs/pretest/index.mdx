## このセクションについて

本番前のリハーサルは、イベント成功の鍵を握る重要なステップです。**良い大会は、本番ではなくリハーサルの質で決まる**と言っても過言ではありません。

---

## 📅 スケジュール概要

| 項目 | 目安期間 | 本番からの逆算 |
|-----|----------|---------------|
| テクニカルリハーサル | 2〜4時間 | 本番 **1〜3日前** |
| 進行リハーサル | 2〜4時間 | 本番 **前日〜当日午前** |
| 最終確認 | 30分〜1時間 | 本番 **直前** |

:::tip[日程例：オフライン大会の場合]
- **本番3日前**: 会場下見・機材搬入
- **本番2日前**: テクニカルリハーサル（配信・音響・映像確認）
- **本番前日**: 進行リハーサル（台本読み合わせ・演出確認）
- **本番当日午前**: ゲネプロ（通しリハーサル）
- **本番1時間前**: 最終確認・スタンバイ
:::

## 📝 リハーサルの詳細

### テクニカルリハーサル（テクリハ）

技術面の動作確認を行うリハーサルです。

| 確認項目 | 内容 |
|---------|------|
| 配信映像 | 正常に出力されているか |
| 音声 | 途切れずに届いているか |
| カメラ・スイッチャー | 想定通りに動作するか |
| ネットワーク | 回線が安定しているか |

:::note[テクリハのポイント]
「問題が起きないこと」を目指すのではなく、**問題が起きた場合にどこで不具合が発生しているのかを把握すること**が重要です。
:::

### 進行リハーサル

香盤表や台本に沿って、実際の大会の流れを確認します。

| 確認項目 | 内容 |
|---------|------|
| タイミング | 司会・キャスターの話し出すタイミング |
| 映像切り替え | 配信開始・画面切り替えの合図 |
| 進行 | 試合開始・休憩に入るタイミング |
| 連携 | 各セクション間の情報共有 |

:::warning[進行リハの目的]
本番通りに完璧に進めることではありません。**問題点を洗い出し、改善していくこと**が重要です。
:::

---

## 💡 リハーサルの考え方

> リハーサルは、「本番を成功させるための準備」であると同時に、**失敗してもよい唯一の時間**でもあります。

- リハーサル中のミスやトラブルは、本番で同じことを起こさないための貴重な材料
- 問題が発生した場合は隠さずに共有
- どのセクションが対応するのかを明確にしながら解決策を検討
